# Restaurante
## Nicolas Alexander Galarraga Piedra
Sistema donde se usan clases para organizar un restaurante usando clases para los productos, clientes y el restaurante en general.

El archivo main.py organiza para llamar a cada uno de los archivos para mostrar sus atributos, la clases Platillo y Bebida reciben atributos para que luego se guarden dentro de la clase Restaurante y se muestren tambien llamando a esta.

Es muy importante modularizar el software ya que mejora completamente la escalabilidad y eficiencia al resolver errores, además al aplicar polimorfismo y encapsulamiento se tiene un mejor control de cada clase
