# Restaurante
## Nicolas Alexander Galarraga Piedra
Sistema donde se usan clases para organizar un restaurante usando clases para los productos, clientes y el restaurante en general.

El archivo main.py organiza para llamar a cada uno de los archivos según que quiera realizar el usuario, la clase producto y cliente reciben atributos para que luego se guarden dentro de la clase Restaurante y se muestren tambien llamando a esta clase

Es muy importante modularizar el software ya que mejora completamente la escalabilidad y eficiencia al resolver errores
